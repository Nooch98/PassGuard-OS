# ⚠️ Legal Notice: Security and Transparency

PassGuard OS is a personal, independent, and open-source project. I am the sole developer and not a professional security engineer.

My primary goal is to provide a transparent, offline tool for managing personal passwords. However, please note the following:

- **No Professional Audit:** While I follow industry-standard security practices (AES-256-GCM, PBKDF2) and prepare the Argon2id implementation, this application has **not** undergone a professional security audit.

- **Experimental Nature:** This is an independent project. It may contain unforeseen bugs, exceptions, or vulnerabilities.

- **Your Responsibility:** Since the source code is public and designed to be compiled by the user, you are responsible for reviewing the code you run on your computer.

- **No Warranty:** This software is provided "as is." Use it first with non-critical data while you verify its stability and security.

### How You Can Help
As I work on this alone, I greatly value community collaboration. If you have experience in cryptography, Flutter development, or security, **your contributions are incredibly valuable.**

- **Found a bug?** Open an issue and let me know.

- **Can you improve security?** Submit a pull request.

- **Discovered a vulnerability?** Follow the instructions in [SECURITY.md](SECURITY.md) to report it publicly so we can all learn and fix it together.

**I'm learning as I develop this.** Your constructive feedback is not only welcome, but it's key to making PassGuard OS better, more secure, and more reliable for everyone. Thank you for your support.
